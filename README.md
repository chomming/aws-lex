## Amazon Lex의 기능
- 대화형 인터페이스를 구축하는 완전 관리형 AI 서비스

## Amazon Lex의 구성요소
- Intent(의도)
  - 사용자가 수행하고자 하는 작업
    - 날씨를 알고 싶다.
- Utterance(발화)
  - 의도를 트리거하는 사용자 입력
    - 오늘 서울 날시 알려줘.
- Slot(슬롯)
  - 수행할 때 필요한 정보
    - 날씨, 지역 등
- Fulfillment(이행)
  - 수행하는 방법
    - AWS Lambda
